# Number-Wizard

The structure of the game states of my project:

1. **__Start Scene__**
 - Two buttons : start and quit
2. **__Core Scene__**
 - Three buttons : highest,lower and correct number
3. **__Win Scene__**
 - Start again button
 
 The scripts i used :
 
 - Number Wizard(Gameplay script)
 - SceneLoader
 
 The fonts I used:
 
 - "Wizards' Magic SDF"
 
  The resources with which I created the game:
